# vim-ace

Vim syntax highlighting for [Ace](https://github.com/yosssi/ace) templates

## Installation

Write below in your init.vim or .vimrc

```
" dein.vim
call dein#add('yosssi/vim-ace')

" plugin.toml
[[plugins]]
repo = 'yosssi/vim-ace'

" NeoBundle
NeoBundle 'yosssi/vim-ace'

" vim-plug
Plug 'yosssi/vim-ace'
```
